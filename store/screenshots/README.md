# Drop App Store screenshots here

Capture from Simulator after `npm run ios` → Run:

1. Features → iPhone 16 Pro Max (or latest 6.9")
2. Navigate to Home, Add expense, Activity, Budgets, Stats
3. File → Save Screen (or ⌘S)
4. Rename clearly, e.g. `01-home.png`

Required size is usually 1320×2868 or 1290×2796 for the largest iPhone slot.
